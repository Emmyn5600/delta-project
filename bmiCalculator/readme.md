
# React Navite app to calculate BMI 






## Run Locally

Clone the project

```bash
  git clone https://github.com/Emmyn5600/delta-project.git
```

Go to the project directory

```bash
  cd bmiCalculator
```

Install dependencies

```bash
  npm install
```

Start the app server

```bash
  npx expo tart
```


## Tech Stack

**Client:**  React-native, custom Css3

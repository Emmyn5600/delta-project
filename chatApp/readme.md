
# Chat App with Node JS and Socket.io






## Run Locally

Clone the project

```bash
  git clone https://github.com/Emmyn5600/delta-project.git
```

Go to the project directory

```bash
  cd chatApp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Tech Stack

**Client:** Html5, Css3

**Server:** Node, Express,socketio

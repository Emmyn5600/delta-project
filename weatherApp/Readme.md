
# Weather App with React JS and OpenWeather Api




## Installation

Install Weather App packages with  npm

```bash
  cd folderName
  
  npm install 
```
    


## Run Locally

Clone the project

```bash
  git clone https://github.com/Emmyn5600/delta-project.git
```

Go to the project directory

```bash
  cd weatherApp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

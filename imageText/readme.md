
# Get Image Text App with Node JS and  OCR technology( tesseract.js)






## Run Locally

Clone the project

```bash
  git clone https://github.com/Emmyn5600/delta-project.git
```

Go to the project directory

```bash
  cd imageText
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Run test

```bash
  npm run test
```


## Tech Stack

**Server:**  Node, tesseract.js
